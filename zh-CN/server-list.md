# 服务器列表

本页面收集了部分可用的服务器列表，点击对应线路后方的**复制**按钮即可复制到剪切板。

如果发现某条线路不可用，请尝试更换线路使用。

## 我应该怎么使用它们？

您应该仔细看一看**使用指南**里的**[快速上手Sonolus](guide/quick-start)**。当然，某些服务器也支持直接在浏览器中打开来查看使用说明。
