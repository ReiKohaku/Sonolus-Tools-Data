# Server list
This page collects a number of available server lists. Click the copy button next to the corresponding server to copy it to your clipboard.

**The official version of Sonolus has been released. It is recommended to use the official coconut server to publish new content. This page is only used for archival purposes and does not guarantee that all links are functional. Thank you for your understanding.**

## How should I use them?
You should take a close look at the **[Quick Start Guide for Sonolus](guide/quick-start** in the user guide. Of course, some servers also support being opened directly in your browser to view the instructions.

## List
If you find that a server is unavailable, please first check your network connection. If there are no issues or conditions preventing access, you may ask others for help. If the issue still cannot be resolved, you can ask in the official server.
